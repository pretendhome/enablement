# Codex Enablement
**Purpose**: Reusable coaching and enablement patterns for helping someone move from reference material to high-trust performance.

This directory is the canonical home for the reusable system.

It is not specific to one interview, one company, or one takehome.
Those are content instances.

The reusable asset is the loop:
- read a small reference
- answer from memory
- grade hard
- refine
- advance only on pass

## Design Principle

Most learning systems over-index on reading and under-index on judged retrieval.

This system optimizes for:
- active recall
- high-bar refinement
- concise, customer-safe phrasing
- progression by demonstrated readiness

## Canonical Pattern

1. Select the round or skill.
2. Read the minimum viable reference.
3. Answer one question live.
4. Grade against a clear bar.
5. Refine the weak point only.
6. Advance only after a pass.
