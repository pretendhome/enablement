# Elia Quickstart

This is the shortest path from zero to a working Palette setup.

## Option A: Claude Project

### Step 1
Go to `claude.ai` and open **Projects**.

### Step 2
Create a new project.

Name it:

```text
Palette
```

### Step 3
Open this file:

[`/home/mical/fde/palette/skills/enablement/enablement-coach.md`](/home/mical/fde/palette/skills/enablement/enablement-coach.md)

Copy the entire file.

### Step 4
Paste it into your Claude Project **Instructions**.

### Step 5
Start a new chat in that project and paste this:

```text
Hi. I want to set up Palette for my real work.

Please help me do this in the shortest possible way.

Start by helping me create my lens:
- who I am
- what I am building
- what tools I use
- what my current bottleneck is this week

Then help me use that lens on one real task immediately.

Do not give me a tour.
Do not explain the whole system.
Propose the next useful move and keep it practical.
```

### Step 6
When the coach reflects your lens back to you, save it in Project Knowledge as:

```text
Elia Lens
```

### Step 7
Then paste this:

```text
Use my lens and help me start the task I actually need this week.

I want one usable output before we stop.

At the end, give me:
1. the artifact we created
2. my updated lens
3. one exact thing to do next in a fresh chat
```

## Option B: Co-Work / Shared Session

If we are doing this together live:

### Step 1
Open Claude Projects.

### Step 2
Create the project.

### Step 3
Paste in:

[`/home/mical/fde/palette/skills/enablement/enablement-coach.md`](/home/mical/fde/palette/skills/enablement/enablement-coach.md)

### Step 4
Paste this starter prompt:

```text
Mical is onboarding me live.

Help us do this in order:
1. build my lens
2. confirm the lens in plain language
3. use it on one real problem from this week
4. give us an updated lens and a short session summary

Keep this plain-language and practical.
```

### Step 5
Do one real task only.

Good first tasks:
- investor update
- product/feature brief
- competitor research
- launch copy
- roadmap decision
- pitch prep

If you want to create the lens manually first, paste this into the chat and fill it in:

```yaml

role: Founder
company_or_project: [dating app name]
what_i_am_building: [one sentence]
daily_tools:
  - [tool]
  - [tool]
  - [tool]
where_my_work_lives: [Google Drive / Notion / local folders / etc.]
current_bottleneck: [what is slowing me down this week]
primary_goal_this_week: [one concrete outcome]
time_budget: [realistic time available]
preferences: [how I like to work / how I want help]
```

## The Only Rule

Do not leave setup without starting a real task.

The session is successful only if you end with:
- a working project
- a saved lens
- one usable output
- one specific next action
